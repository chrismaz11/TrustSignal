# TrustSignal — Vanta Demo Video

**Status: `marketing` `non-product`**

This is a marketing demo asset built with [Remotion](https://remotion.dev) and [Code Hike](https://codehike.org). It renders animated code walkthrough videos used for sales, investor, and partner presentations.

**This directory is not part of the TrustSignal product.** It contains no production logic, API code, or compliance-sensitive components.

---

## Usage

```console
npm install
npm run dev        # preview in browser
npx remotion render  # render to video file
```

Code snippets used in the video are in the `public/` folder.
